GAPIC library for the Google Datastore API
=============================================================================================================

This is the API documentation for ``gapic-google-datastore-v1``.

gapic-google-datastore-v1 uses google-gax_ (Google API extensions) to provide an
easy-to-use client library for the `Google Datastore API`_ (v1) defined in the googleapis_ git repository


.. _`google-gax`: https://github.com/googleapis/gax-python
.. _`googleapis`: https://github.com/googleapis/googleapis/tree/master/google/google&#x2F;datastore/v1
.. _`Google Datastore API`: https://developers.google.com/apis-explorer/?hl=en_US#p/datastore/v1/
